# 用户手册 (User Manual)

OfficeAgent 是一个可视化的 AI 工作流编排平台，允许您通过拖拽组件来创建复杂的多模态任务。本文档将指导您如何使用该平台。

## 1. 界面概览

OfficeAgent 的主界面由三个主要部分组成：

1.  **左侧节点面板 (Sidebar)**：包含可用的节点类型（Input, LLM, Tool, Output），可直接拖拽到画布。
2.  **中间画布 (Canvas)**：这是您的工作区，您可以在这里放置、连接和配置节点。
3.  **顶部工具栏 (Header)**：包含保存、加载、调试等功能按钮。

## 2. 节点详解

OfficeAgent 目前支持四种类型的节点：

### 2.1 输入节点 (Input Node)
*   **功能**：作为工作流的起点，接收用户的文本输入。
*   **配置**：
    *   **变量名**：默认为 `user_input`。
    *   **类型**：String（文本）。
    *   **描述**：对输入的说明。

### 2.2 大模型节点 (LLM Node)
*   **功能**：调用大语言模型（如 DeepSeek 或通义千问）生成文本。
*   **配置**：
    *   **模型选择**：选择您希望使用的模型（DeepSeek/Qwen）。
    *   **API Key**：输入对应模型的 API 密钥。
    *   **API Base URL**：如果使用 DeepSeek，通常为 `https://api.deepseek.com`。
    *   **温度 (Temperature)**：控制生成内容的随机性（0.0-1.0）。
    *   **提示词模板 (Prompt)**：定义模型的系统提示词。您可以使用 `{{input}}` 或其他上游节点的输出作为变量。

### 2.3 工具节点 (Tool Node / TTS)
*   **功能**：执行特定的工具操作，目前主要是 **文本转语音 (TTS)**。
*   **配置**：
    *   **API Key**：阿里云 DashScope 的 API Key。
    *   **模型**：如 `cosyvoice-v1` 或 `qwen3-tts-flash`。
    *   **文本 (Text)**：选择“引用变量”以使用 LLM 节点的输出文本。
    *   **音色 (Voice)**：选择喜欢的声音（如 Cherry, Serena, Ethan）。
    *   **语言类型**：默认为 Auto。

### 2.4 输出节点 (Output Node)
*   **功能**：作为工作流的终点，定义最终返回给用户的数据。
*   **配置**：
    *   **输出变量**：可以定义多个输出字段（如 `voice_url`, `text_content`）。
    *   **引用**：选择要输出的上游节点变量。

## 3. 创建与编辑工作流

### 3.1 添加节点
从左侧面板拖拽所需的节点到画布中。每个节点都会自动分配一个唯一的 ID。

### 3.2 连接节点
将鼠标移动到节点的右侧圆点（输出句柄），按住并拖动到另一个节点的左侧圆点（输入句柄）。连接线表示数据流向。

### 3.3 配置参数
点击任意节点，右侧会弹出配置抽屉。在这里您可以输入 API Key、提示词等详细信息。修改完成后点击“应用”保存。

## 4. 调试与运行

点击顶部工具栏的 **“调试”** 按钮，打开调试面板：

1.  **输入测试数据**：在“用户输入”框中填写测试文本。
2.  **开始执行**：点击“开始执行”按钮。
3.  **查看日志**：下方会实时显示执行日志。
    *   **节点状态**：每个节点会从“等待”变为“执行中”，最后变为“成功”或“失败”。
    *   **进度条**：对于耗时较长的 TTS 节点，会显示实时的生成进度（如“正在生成第 5/20 片段”）。
4.  **结果展示**：如果工作流生成了音频，调试面板底部会自动出现音频播放器，您可以直接播放。

## 5. 常见问题

### Q: 为什么调试时一直显示“加载中”？
A: 检查 RabbitMQ 服务是否启动。后端依赖消息队列来分发任务，如果连接失败，任务将无法执行。

### Q: TTS 生成失败，提示 API Key 无效？
A: 请确保您在 Tool 节点的配置中填入了正确的阿里云 DashScope API Key，并且该 Key 有权访问 CosyVoice 服务。

### Q: 长文本生成音频会超时吗？
A: 不会。OfficeAgent 使用了异步执行机制，支持长达 10 分钟以上的任务执行时间。您可以放心生成长篇播客。
